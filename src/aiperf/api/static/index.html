AIPerf API

Real-time benchmark monitoring via HTTP and WebSocket

GET /dashboard

Live dashboard with real-time progress and metrics via WebSocket.

HTTP Endpoints

GET /metrics

Prometheus Exposition Format metrics for scraping. Compatible with PodMonitors and Prometheus.

GET /api/metrics

JSON metrics with all current benchmark measurements.

GET /api/status

Current benchmark status (state, phase, profile ID).

GET /api/progress

Current progress (requests completed, total, percent).

GET /api/workers

Worker status summary (total, active, per-worker status).

GET /api/config

Current benchmark configuration as JSON.

GET /health

Health check endpoint. Returns "ok" when the API is running.

WebSocket Endpoint

WS /ws

Real-time ZMQ message stream with dynamic subscriptions.

// Subscribe to message types
ws.send(JSON.stringify({
    type: "subscribe",
    message_types: ["realtime_metrics", "credit_phase_progress"]
}));

// Receive messages
ws.onmessage = (event) => {
    const msg = JSON.parse(event.data);
    console.log(msg.message_type, msg);
};

Usage Examples

Prometheus Scraping

curl http://localhost:9090/metrics

JSON Metrics

curl http://localhost:9090/api/metrics | jq

WebSocket Client (Python)

import asyncio
import websockets
import json

async def stream_metrics():
    async with websockets.connect("ws://localhost:9090/ws") as ws:
        await ws.send(json.dumps({
            "type": "subscribe",
            "message_types": ["realtime_metrics"]
        }))
        async for message in ws:
            print(json.loads(message))

asyncio.run(stream_metrics())
